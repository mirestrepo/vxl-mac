This package contains the Manchester public not-quite VXL library.
Current contents
v1l/io  Binary IO functions for the level 1 libraries.
vsl     Binary IO support for VXL
We are planning to promote the above libraries to VXL as soon as possible.

mbl	  Some basic stuff needed by the Manchester libraries.

Expected Contents

vpdfl   Multivariate PDF objects, builders, samplers, etc.
