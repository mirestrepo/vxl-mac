A. Filenames and structure

The directory structure of VXL is below
 
  $VXLROOT/vxl/vcl
              /vnl
              /vbl
              /vil
              /config.$BUILD
          /config
          /Scripts
The last two directories are part of the IUE/TargetJr makefiles so are not needed by those using Visual Studio or their own build systems.
